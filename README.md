# Pokédex App

A simple and interactive Pokédex built using the [PokéAPI](https://pokeapi.co/).  
This app allows users to search for Pokémon and view detailed information including their image, types, abilities, stats, and moves.

## Features

- Search Pokémon by name or ID
- Display Pokémon sprite (image)
- Show Pokémon types and abilities
- Display detailed stats (HP, Attack, Defense, etc.)
- Scrollable list of moves
- Responsive and clean design

## Technologies Used

- *Frontend:* React.js
- *API:* [PokéAPI](https://pokeapi.co/)
- *Styling:* CSS
